# Review UX (prep)

Design-only specs for device-centric Review (006C). **No TypeScript** under freeze.

## Required surfaces

1. Device table (`device_id`, SN, TID, SIM, image count, match, status)  
2. Image pane — only selected device `images[]`  
3. Fields pane — per-field confidence  
4. Match pane — `matched_reason[]` checklist  
5. Feedback capture — store corrections for Golden Dataset / 006E  

Wireframes / exported notes may live here (`wireframes.md`, links to Figma).
